# PetzKare

PetzKare is a premium e-commerce application for pet supplies, offering a wide range of products including food, treats, toys, and accessories for dogs and cats.

## Features

- **Shop by Category**: Easily browse products for dogs and cats.
- **Search & Filter**: Find exactly what you need with powerful search and filter options.
- **Cart & Checkout**: Seamless shopping experience.
- **Premium Design**: Modern, responsive UI with vibrant aesthetics.

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router

## Getting Started

1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev` to start the local server.
